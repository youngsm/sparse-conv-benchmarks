# Sparse-convolution library benchmark on PILArNet-M

- **Device:** NVIDIA H200 NVL
- **torch / CUDA:** 2.5.1+cu121 / cu12.1
- **Precision:** bf16 (autocast; TF32 enabled for fp32 matmuls)
- **spconv:** spconv-cu124 2.3.8
- **torchsparse:** torchsparse 2.1.0
- **warpconvnet:** warpconvnet 1.7.11
- **minkowski:** MinkowskiEngine 0.5.4
- **Model sizes (params):** small=1.5M, medium=10.6M, large=61.1M

Rows are `(network, batch size, active input voxels)`; columns are libraries.
